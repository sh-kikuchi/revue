## 三角形を作ろう。

CSSで三角形を作ってみましょう。スライドを作るときに<br>
矢印としても活用できます。

■ HTML

```
<div class="triangle"></div>

```

■ CSS
```
.triangle{
  width: 0px;
  border-color: tomato; //ここは好きな色♪
  border-style: solid;
  border-width: 30px;
  border-left-color: transparent;
  border-top-color: transparent;
  border-right-color: transparent;
}
```
